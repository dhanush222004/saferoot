# Saferoot: AI Herbal Supply Chain

Welcome to **Saferoot**, an AI-powered solution for Herbal Supply Chain management.

This repository contains everything you need to run your app locally and manage herbal products effectively from farmers to customers.

## Overview

Saferoot provides real-time tracking, quality assessment, and AI-driven insights to ensure transparency and quality in the herbal supply chain.

## Features

- **Role-Based Access**: Specialized dashboards for Farmers, Labs, Factories, and Customers.
- **Product Journey Tracking**: Trace herbs from harvest to final product.
- **Verification & Authentication**: Built-in verification mechanisms for supply chain integrity.
- **Bio-Waste Tracking**: Efficient routing and tracking of bio-waste.
- **AI Chatbot**: Intelligent assistant for answering queries.

## Run Locally

**Prerequisites:**  Node.js (v18+)

1. Install dependencies:
   ```bash
   npm install
   ```
2. Environment Configuration:
   Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key.
3. Start the development server:
   ```bash
   npm run dev
   ```

## Production Build

To build the application for production:
```bash
npm run build
```
